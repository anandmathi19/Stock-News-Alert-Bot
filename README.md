# 📈 Stock News Alert Bot – Python

A Python automation project that monitors stock price changes and sends news alerts when significant price movements occur. The bot fetches stock data and related news using APIs and notifies the user with the latest updates.

---

## 🚀 Features

- Monitors stock price changes
- Fetches latest news related to the stock
- Sends alerts when major price fluctuations occur
- Automated data retrieval using APIs

---

## 🛠 Technologies Used

- Python
- Stock Price API
- News API
- Requests Library
